## 📌 About The Project

HireAI is a complete AI-powered recruitment management platform built for HR teams and job candidates. It streamlines the entire hiring process from candidate application to final selection, while also helping candidates prepare for interviews using an AI Voice Assistant.

---

## ✨ Features

### 👨‍💼 Admin Dashboard
- View all HR accounts and candidates
- Recruitment analytics and hiring trends
- Department-wise hiring reports
- Candidate performance comparison

### 👩‍💼 HR Dashboard
- Add and manage candidates
- Interview scheduling with calendar
- Candidate status tracking pipeline
- Resume database management
- Real-time candidate search

### 🧑‍💻 Candidate Portal
- Register and login securely
- Upload resume (PDF/DOC)
- View interview schedule
- Track application status
- View skill analysis

### 🤖 AI Mock Interview
- Voice-based interview questions
- Speech recognition (microphone input)
- Real AI feedback powered by OpenAI
- 4 interview modes: HR, Technical, Communication, Aptitude
- Confidence, Communication and Technical scores
- Suggestions for improvement

## 🚀 Getting Started Locally

### Prerequisites
- Node.js (v18 or higher)
- MongoDB (local or Atlas)
- Git

### Installation

**1. Clone the repository:**
```bash
git clone https://github.com/jrithika305/recruitment-system.git
cd recruitment-system
```

**2. Install backend dependencies:**
```bash
cd backend
npm install
```

**3. Create `.env` file in backend folder:**

**4. Start the backend server:**
```bash
node server.js
```

**5. Open frontend:**
- Open `frontend/index.html` with Live Server in VS Code
